Simulador de marchas

Proyecto de práctica en HTML, CSS y JavaScript que simula el funcionamiento básico de un vehículo. Incluye acciones como arrancar y parar el motor, cambiar de marchas (1ª a 4ª y marcha atrás), reproducir sonido y modificar datos dinámicos como el piloto actual.

El diseño visual está inspirado en un estilo pixel art retro, con una interfaz clara y botones interactivos.
