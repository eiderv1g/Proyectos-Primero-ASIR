# Tipos de aplicaciones, licencias y software de propósito general

## 1. SOFTWARE DEL SISTEMA

Es el **software básico que permite que el ordenador funcione** y que el hardware pueda usarse.

### Sistema Operativo (SO)

Es el programa principal que **gestiona el hardware y permite ejecutar aplicaciones**. Sus funciones son:

- Controlar memoria
- Gestionar archivos
- Ejecutar programas
- Administrar dispositivos

### Controladores (Drivers)

Son programas que **permiten al sistema operativo comunicarse con el hardware**.

Ejemplos:

- Driver de la impresora
- Driver de la tarjeta gráfica
- Driver del teclado

Sin drivers, el sistema operativo **no sabría usar los dispositivos**.

---

## 2. SOFTWARE DE APLICACIÓN

Son programas creados para **realizar tareas específicas para el usuario**.

### Procesadores de texto

Sirven para **crear y editar documentos.**

*Ejemplos: Microsoft Word, Google Docs*

### Navegadores web

Permiten **acceder y navegar por Internet**.ç

*Ejemplos: Mozilla Firefox, Chrome, Microsoft Edge*

### Hojas de cálculo

Sirven para **trabajar con datos, cálculos y gráficos**

*Ejemplos: Excel, Google Sheets*

### Programas de diseño gráfico

Se usan para **crear o editar imágenes y gráficos**.

*Ejemplos: Photoshop, GIMP, Canva*

---

## 3. LICENCIAS DE SOFTWARE

Una **licencia de software** es el **permiso legal que indica cómo se puede usar un programa**.

| Tipo de licencia | Precio | ¿Se puede modificar el código? | Características | Ejemplos |
| --- | --- | --- | --- | --- |
| **Software Propietario** | Normalmente de pago | ❌ No | El software tiene dueño y su código no es público. El usuario solo puede usarlo según las condiciones de la licencia. | Windows, Microsoft Office, Photoshop |
| **Freeware** | ✅ Gratis | ❌ No | Se puede usar gratuitamente, pero el código no se puede modificar ni redistribuir libremente. | Adobe Acrobat Reader, Skype |
| **Shareware** | Prueba gratis | ❌ No | Permite usar el programa durante un tiempo limitado o con funciones restringidas; luego hay que pagar. | WinRAR, WinZip |
| **Software Libre (FOSS)** | Normalmente gratis | ✅ Sí | El código es abierto y permite usar, estudiar, modificar y distribuir el programa. | Linux, LibreOffice, GIMP |
- **Propietario → pagas y no puedes tocar el código**
- **Freeware → gratis pero cerrado**
- **Shareware → prueba temporal**
- **Software libre → código abierto y modificable**
